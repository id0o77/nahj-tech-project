# Nahj Tech Website

## فكرة المشروع
موقع تعريفي لشركة تقنية يهدف إلى عرض الخدمات بشكل احترافي، مع توفير لوحة تحكم لإدارة الخدمات، بالإضافة إلى نظام تواصل، تقييم العملاء، وطلب الخدمات.

## المميزات
- تصميم حديث ومتجاوب (Responsive)
- دعم عربي / English مع تحويل الاتجاه RTL و LTR
- لوحة تحكم لإدارة الخدمات (إضافة / تعديل / حذف)
- نظام طلب خدمات مرتبط بكل خدمة
- نظام تقييم العملاء (نجوم + رضا)
- حفظ رسائل التواصل في قاعدة البيانات
- صفحة اقتراحات وآراء العملاء
- حماية (CSRF، تشفير كلمات المرور، حماية الجلسات)

## طريقة التشغيل

```bash
python -m pip install -r requirements.txt
python app.py
```

ثم افتح:
http://127.0.0.1:5000
